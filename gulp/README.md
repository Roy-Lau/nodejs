# mpa_cli

### 为什么做？

> 现在前端虽然是 spa(单页应用)的天下了。但是,还有一些固执的公司还在用多页。所以 mpa-cli 横空出世了！

### 做成什么样？

1. 将 HTML 从开发目录移动到生产目录
2. 如果用的是 less/sass/styl 等将其编译成 css，如果用的是 css 对其压缩并自动加入浏览器前缀
3. 压缩打包图片和字体图标资源
4. 压缩混淆 js,将新的 ES6/7 语法转化成 ES5
5. 使用热更新,文件保存后页面自动更新
6. 做个反向代理,解决跨域问题
